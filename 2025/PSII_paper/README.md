# PSII Paper Analysis Scripts

This directory contains scripts used for the analysis of Photosystem II (PSII) simulations, specifically related to MCCE (Multi-Conformation Continuum Electrostatics) calculations.

## Scripts in `mcce_neededFiles`

The `mcce_neededFiles` directory contains the following Python scripts:

- **`McceProtienAnalyzer.py`**: Tools for analyzing MCCE protein output.
- **`ProtonationProfileVisualizer.py`**: Visualization tools for protonation profiles.
- **`analysis_pKa_distribution.py`**: Script to analyze pKa distributions.
- **`extract_conformers_step2out.py`**: Extracts conformers from MCCE step 2 output.
- **`fitpka.py`**: Utility for fitting pKa values.
- **`mcce2extendedPDB_converter.py`**: Converts MCCE output to extended PDB format.
- **`psii_cofactor_charges_frank.py`**: Manages cofactor charges for PSII.
- **`rename_non_standard_2_standard.py`**: Renames non-standard residues to standard ones.
- **`run_mcce2Extended_conversion_terminal.py`**: Terminal runner for MCCE to extended PDB conversion.
